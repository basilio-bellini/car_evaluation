# 🚗 Оценка автомобилей на основе технологий больших данных

Дипломный проект по разработке системы автоматической оценки рыночной стоимости 
автомобилей с использованием машинного обучения и технологий больших данных.

## 📋 Описание

Система собирает данные об объявлениях о продаже автомобилей с портала Auto.ru,
обучает модель машинного обучения и предоставляет веб-интерфейс для оценки
рыночной стоимости автомобиля по его характеристикам.

## 🛠️ Стек технологий

- **Парсинг данных** — Playwright, BeautifulSoup4
- **Обработка данных** — Pandas, NumPy
- **Машинное обучение** — Scikit-learn / CatBoost
- **Веб-приложение** — Streamlit
- **Хранение данных** — CSV / SQLite

## 🚀 Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/username/auto-price-estimator.git
cd auto-price-estimator
```

### 2. Установить зависимости
```bash
pip install -r requirements.txt
playwright install chromium
```

### 3. Собрать данные
```bash
python parser/scraper.py
```

### 4. Обучить модель
```bash
python model/train.py
```

### 5. Запустить приложение
```bash
streamlit run app/main.py
```
