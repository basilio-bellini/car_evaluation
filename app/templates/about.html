АвтоРазум — умная оценка автомобилей

Информационная система для автоматической оценки рыночной стоимости автомобилей на основе технологий машинного обучения. Система анализирует тысячи объявлений с авторынка и предсказывает справедливую цену по характеристикам автомобиля.

Описание проекта

АвтоРазум — дипломный проект по разработке информационной системы оценки стоимости автомобилей с использованием технологий больших данных и машинного обучения.

Система решает актуальную задачу для покупателей и продавцов автомобилей — определение справедливой рыночной цены без необходимости вручную анализировать сотни объявлений. Достаточно указать характеристики автомобиля и система мгновенно выдаст оценку на основе реальных рыночных данных.

Данные для обучения модели собраны с крупнейшего российского автомобильного портала Auto.ru с использованием технологий веб-скрапинга.

Методология оценки
1
Сбор данных

Автоматический сбор объявлений с Auto.ru с помощью Playwright. Собрано более 100 000 объявлений о продаже автомобилей с пробегом.

2
Предобработка данных

Очистка данных, удаление дубликатов, инженерия числовых и категориальных признаков.

3
Обучение модели

Сравнение нескольких алгоритмов машинного обучения. Выбор лучшей модели на основе метрик MAE, RMSE и R².

4
Оптимизация

Подбор гиперпараметров с помощью Optuna для достижения наилучшего качества предсказаний.
